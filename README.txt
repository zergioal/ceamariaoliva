Proyecto: Sitio Web Institucional - C.E.A. Madre María Oliva

Desarrollado por: Javier Enrique Delgadillo
Fecha: 07 de julio de 2025

Descripción:
Este sitio web es una representación institucional informativa de las distintas carreras técnicas ofrecidas por el C.E.A. Madre María Oliva. Fue desarrollado como parte de un proyecto técnico y es donado a la institución para su uso libre y futura personalización.

Estructura del sitio:
- index.html — Página principal con presentación institucional
- *.html — Subpáginas individuales por carrera
- /styles — Hojas de estilo CSS
- /scripts — Archivos JavaScript del sitio
- /Images — Imágenes generales del sitio
- /Images2 — Imágenes adicionales u organizadas por secciones

Funcionalidad destacada:
- En la carpeta /scripts se incluye un archivo llamado mensaje.js, el cual permite generar un enlace funcional de WhatsApp con un mensaje predeterminado. Este script se utiliza para facilitar el contacto inmediato con la institución desde la web, redirigiendo al usuario a la app o versión web de WhatsApp con el texto ya escrito.

Cómo usar:
1. Descomprimir el archivo .zip.
2. Abrir el archivo index.html en cualquier navegador moderno.
3. Mantener la estructura de carpetas sin mover los archivos, para que estilos, imágenes y scripts funcionen correctamente.

Requisitos:
- El sitio puede funcionar sin conexión a internet para navegación básica (secciones, imágenes, diseño).
- El botón de contacto vía WhatsApp requiere conexión a internet para redirigir correctamente al servicio.
